# Sandbox Wise Transfer APL

A sandbox environment for the Wise Transfer API Learning platform (APL).

## Overview

This repository contains a sandbox implementation for testing and developing with the Wise Transfer API.

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Building

```bash
npm run build
```

### Development

```bash
npm run dev
```

## API Reference

For more information about the Wise Transfer API, visit the official documentation.
